# ADR-007: File Uploads via Cloudflare R2

**Date:** 2025-08-01
**Status:** Accepted

## Decision
Store candidate-uploaded files (CVs, ID photos) in Cloudflare R2, referenced by key in the `file_uploads` table.

## Rationale
R2 has zero egress fees, direct Worker binding, and S3-compatible API. Files are scoped under `uploads/{tenantId}/{formId}/{responseId}/`. Serving files through the Worker allows auth-gated download in the future.

## Consequences
Workers have a 100 MB request body limit. Max upload is capped at 10 MB per file. For larger files in the future, use multipart upload via the S3-compatible API with presigned URLs.
