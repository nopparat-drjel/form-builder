# ADR-005: Neumorphic Design System with Tailwind v4

**Date:** 2025-08-01
**Status:** Accepted

## Decision
Implement a custom neumorphic design language using Tailwind CSS v4 extended with green-900/700/500 brand colors and box-shadow tokens (neu, neu-sm, neu-xs, neu-in, neu-in-sm).

## Rationale
Neumorphism gives HR FormKit a premium, modern look appropriate for a Thai B2B SaaS product. The 8px shadow grid stays consistent. Prompt and Sarabun fonts support Thai typography correctly.

## Consequences
Neumorphic shadows do not meet WCAG contrast on every surface. Supplement with clear focus rings and sufficient text contrast. Shadow values are hardcoded to the `#f5f5f5` background — do not place neumorphic components on other background colors.
