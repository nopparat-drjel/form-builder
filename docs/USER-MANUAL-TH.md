## 1. การเข้าสู่ระบบ

### ขั้นตอนการเข้าสู่ระบบ

1. เปิดเบราว์เซอร์ (แนะนำ Google Chrome หรือ Microsoft Edge เวอร์ชันล่าสุด)
2. พิมพ์ที่อยู่เว็บไซต์: **https://hr-form-frontend.vercel.app/login**
3. กรอกข้อมูลดังนี้:
   - **อีเมล:** admin@hrformkit.com
   - **รหัสผ่าน:** Admin@1234
4. คลิกปุ่ม **"เข้าสู่ระบบ"**
5. ระบบจะพาไปยังหน้า Dashboard โดยอัตโนมัติ

> **หมายเหตุ:** ระบบจะจดจำสถานะเข้าสู่ระบบของท่านไว้โดยอัตโนมัติ หากไม่ได้ใช้งานเป็นเวลานาน ระบบจะต่ออายุ session ให้อัตโนมัติ ท่านไม่จำเป็นต้องเข้าสู่ระบบใหม่ทุกครั้ง

> **คำแนะนำด้านความปลอดภัย:** อย่าแชร์ข้อมูลเข้าสู่ระบบกับผู้อื่น และหลีกเลี่ยงการเข้าสู่ระบบบนคอมพิวเตอร์สาธารณะ

## 2. หน้า Dashboard

หน้า Dashboard คือหน้าหลักที่ท่านจะเห็นหลังเข้าสู่ระบบ ออกแบบมาเพื่อให้ท่านเห็นภาพรวมของระบบในทันที

### ส่วนประกอบของ Dashboard

#### การ์ดสถิติ (Stats Cards)

ด้านบนของหน้าจะแสดงการ์ดสถิติ 3 ใบ ได้แก่:

| การ์ด | ความหมาย |
|---|---|
| ใบสมัครทั้งหมด | จำนวนใบสมัครที่ได้รับสะสมทั้งหมดในระบบ |
| ใบสมัครใหม่วันนี้ | จำนวนใบสมัครที่ส่งเข้ามาในวันนี้ |
| แบบฟอร์มทั้งหมด | จำนวนแบบฟอร์มที่สร้างไว้ในระบบ |

#### รายการใบสมัครล่าสุด (Recent Responses)

- แสดงใบสมัคร **5 รายการล่าสุด** ที่ส่งเข้ามา
- ท่านสามารถคลิกที่รายการใดก็ได้เพื่อดูรายละเอียดใบสมัครนั้น
- คลิกไอคอนดาว (★) เพื่อติดดาวใบสมัครสำคัญ

#### Quick Action Cards

ด้านล่างแสดงปุ่มลัด 2 ปุ่ม:

- **"สร้างแบบฟอร์ม"** — คลิกเพื่อเริ่มสร้างแบบฟอร์มใหม่ทันที
- **"ดูใบสมัคร"** — คลิกเพื่อไปยังหน้ารายการใบสมัครทั้งหมด

#### Onboarding Banner

หากท่านยังไม่มีแบบฟอร์มในระบบ จะมีแบนเนอร์แนะนำปรากฏขึ้น คลิกปุ่ม **"สร้างแบบฟอร์มแรกของคุณ"** เพื่อเริ่มต้นใช้งาน

## 3. การจัดการแบบฟอร์ม

เมนู **"แบบฟอร์ม"** ในแถบด้านซ้ายจะพาท่านไปยังหน้ารายการแบบฟอร์มทั้งหมด

### 3.1 ดูรายการแบบฟอร์ม

หน้านี้จะแสดงแบบฟอร์มทั้งหมดที่สร้างไว้ พร้อมข้อมูล ดังนี้:

- ชื่อแบบฟอร์มและคำอธิบาย
- สถานะ: **เปิดใช้งาน** (สีเขียว) หรือ **ปิดใช้งาน** (สีเทา)
- จำนวนใบสมัครที่ได้รับ
- วันที่สร้าง

### 3.2 สร้างแบบฟอร์มใหม่

1. คลิกปุ่ม **"+ สร้างแบบฟอร์ม"** มุมขวาบน
2. กรอกข้อมูลในหน้าต่างที่ปรากฏขึ้น:
   - **ชื่อแบบฟอร์ม** (จำเป็น): เช่น "ใบสมัครงานตำแหน่ง Marketing Officer"
   - **คำอธิบาย** (ไม่บังคับ): เช่น "แบบฟอร์มสำหรับรอบการสมัครเดือนมีนาคม 2569"
3. คลิกปุ่ม **"สร้าง"**
4. ระบบจะสร้างแบบฟอร์มและเปิด Form Builder ให้ท่านแก้ไขทันที

### 3.3 เปิด/ปิดใช้งานแบบฟอร์ม

- คลิกสวิตช์ Toggle ที่อยู่ถัดจากชื่อแบบฟอร์ม
- **สถานะ "เปิด":** ผู้สมัครสามารถเข้าถึงและกรอกแบบฟอร์มได้
- **สถานะ "ปิด":** ลิงก์แบบฟอร์มจะไม่สามารถใช้งานได้ชั่วคราว

> **เคล็ดลับ:** ปิดแบบฟอร์มเมื่อหมดรอบการรับสมัครเพื่อไม่ให้มีใบสมัครเข้ามาเพิ่ม โดยข้อมูลใบสมัครเดิมจะยังคงอยู่ในระบบ

### 3.4 แก้ไขแบบฟอร์ม

คลิกปุ่ม **"แก้ไข"** หรือคลิกที่ชื่อแบบฟอร์ม เพื่อเข้าสู่ Form Builder

### 3.5 ลบแบบฟอร์ม

1. คลิกปุ่ม **"ลบ"** (ไอคอนถังขยะ) ที่อยู่ในแถวของแบบฟอร์มที่ต้องการลบ
2. ระบบจะแสดงหน้าต่างยืนยัน เพื่อป้องกันการลบโดยไม่ตั้งใจ
3. คลิก **"ยืนยันการลบ"** เพื่อดำเนินการ

> **คำเตือน:** การลบแบบฟอร์มจะลบข้อมูลทั้งหมดที่เกี่ยวข้อง รวมถึงใบสมัครทั้งหมดที่ส่งเข้ามา การกระทำนี้ไม่สามารถยกเลิกได้

## 4. การสร้างและแก้ไขแบบฟอร์ม (Form Builder)

Form Builder คือเครื่องมือสร้างแบบฟอร์มแบบ Drag-and-Drop ที่ออกแบบมาให้ใช้งานง่าย โดยไม่ต้องมีความรู้ด้านเทคนิค

### 4.1 โครงสร้างหน้า Form Builder

หน้า Form Builder แบ่งออกเป็น 3 ส่วน:

| ส่วน | ตำแหน่ง | หน้าที่ |
|---|---|---|
| Block Palette (คลังบล็อก) | ซ้ายมือ | รายการบล็อกทุกประเภทที่ใช้สร้างแบบฟอร์มได้ |
| Canvas (พื้นที่ออกแบบ) | กลาง | แสดงแบบฟอร์มที่กำลังสร้าง ลากวางบล็อกที่นี่ |
| Settings (การตั้งค่า) | ขวามือ | ปรับแต่งบล็อกที่เลือกอยู่ |

### 4.2 บล็อก 11 ประเภท

#### กลุ่มเนื้อหา (Content)

| ประเภทบล็อก | การใช้งาน |
|---|---|
| **Heading (หัวข้อ)** | ใส่หัวข้อหลักหรือหัวข้อย่อยในแบบฟอร์ม เช่น "ข้อมูลส่วนตัว" หรือ "ประวัติการศึกษา" |
| **Paragraph (ย่อหน้า)** | ใส่คำอธิบาย คำชี้แจง หรือข้อความแจ้งเตือน เช่น "กรุณากรอกข้อมูลให้ครบถ้วนและถูกต้อง" |

#### กลุ่มช่องกรอกข้อมูล (Input Fields)

| ประเภทบล็อก | การใช้งาน | ตัวอย่าง |
|---|---|---|
| **Short Answer (คำตอบสั้น)** | สำหรับข้อความสั้นๆ 1 บรรทัด | ชื่อ-นามสกุล, ตำแหน่งที่สมัคร |
| **Long Answer (คำตอบยาว)** | สำหรับข้อความหลายบรรทัด | ประวัติย่อ, เหตุผลที่สนใจสมัคร |
| **Email (อีเมล)** | สำหรับกรอกที่อยู่อีเมล พร้อม validation อัตโนมัติ | อีเมลสำหรับติดต่อ |
| **Phone (โทรศัพท์)** | สำหรับกรอกเบอร์โทรศัพท์ | เบอร์โทรศัพท์มือถือ |
| **Number (ตัวเลข)** | สำหรับกรอกข้อมูลที่เป็นตัวเลขเท่านั้น | อายุ, เงินเดือนที่คาดหวัง |
| **Date (วันที่)** | สำหรับเลือกวันที่จากปฏิทิน | วันเกิด, วันที่เริ่มงานได้ |

#### กลุ่มตัวเลือก (Choice Fields)

| ประเภทบล็อก | การใช้งาน | ตัวอย่าง |
|---|---|---|
| **Dropdown (รายการแบบเลื่อนลง)** | เลือก 1 ตัวเลือกจากรายการ เหมาะสำหรับรายการยาว | จังหวัด, ระดับการศึกษา |
| **Multiple Choice / Radio (ตัวเลือกเดี่ยว)** | เลือกได้เพียง 1 ตัวเลือก แสดงทุกตัวเลือกพร้อมกัน | เพศ, ประเภทงาน (Full-time / Part-time) |
| **Checkboxes (ช่องทำเครื่องหมาย)** | เลือกได้หลายตัวเลือกพร้อมกัน | ทักษะที่มี, ภาษาที่ใช้ได้ |

#### กลุ่มไฟล์

| ประเภทบล็อก | การใช้งาน | ตัวอย่าง |
|---|---|---|
| **File Upload (อัปโหลดไฟล์)** | ให้ผู้สมัครแนบไฟล์ | Resume, รูปถ่าย, เอกสารประกอบ |

### 4.3 การเพิ่มบล็อกลงในแบบฟอร์ม

**วิธีที่ 1: ลากและวาง (Drag & Drop)**
1. ชี้เมาส์ไปที่บล็อกที่ต้องการใน Block Palette ด้านซ้าย
2. กดค้างและลากมาวางในตำแหน่งที่ต้องการใน Canvas

**วิธีที่ 2: คลิกเพื่อเพิ่ม**
1. คลิกที่บล็อกใน Block Palette
2. บล็อกจะถูกเพิ่มต่อท้ายแบบฟอร์มโดยอัตโนมัติ

### 4.4 การจัดเรียงบล็อก

- ชี้เมาส์ไปที่บล็อกใน Canvas จะเห็นไอคอนจับ (handle) ปรากฏขึ้น
- กดค้างที่ไอคอนจับและลากบล็อกไปยังตำแหน่งใหม่

### 4.5 การตั้งค่าบล็อก

1. คลิกที่บล็อกใดก็ได้ใน Canvas
2. แผงการตั้งค่า (Settings) ด้านขวาจะอัปเดตเพื่อแสดงตัวเลือกของบล็อกนั้น

**ตัวเลือกที่ตั้งค่าได้:**

| การตั้งค่า | ความหมาย |
|---|---|
| Label (ชื่อฟิลด์) | ข้อความคำถามที่แสดงให้ผู้สมัครเห็น |
| Placeholder (ข้อความตัวอย่าง) | ข้อความแนะนำที่แสดงในช่องกรอก เช่น "กรอกชื่อจริง" |
| Required (จำเป็นต้องกรอก) | เปิด/ปิด การบังคับกรอกข้อมูลในช่องนี้ |
| ตัวเลือก (Options) | สำหรับ Dropdown, Radio, Checkboxes — เพิ่ม แก้ไข หรือลบตัวเลือก |

> **เคล็ดลับ:** ช่องที่มีเครื่องหมาย * สีแดง คือช่องที่ตั้งค่าเป็น "Required" ผู้สมัครจะต้องกรอกก่อนจึงจะส่งแบบฟอร์มได้

### 4.6 การอัปโหลดโลโก้องค์กร

1. ใน Canvas ให้คลิกที่พื้นที่โลโก้ด้านบนสุดของแบบฟอร์ม
2. เลือกไฟล์รูปภาพจากเครื่องของท่าน (รองรับ JPG, PNG, WebP)
3. ระบบจะปรับขนาดรูปโดยอัตโนมัติให้เหมาะสม (400 x 200 พิกเซล)

### 4.7 การบันทึกแบบฟอร์ม

- ระบบ **บันทึกอัตโนมัติ** ทุกครั้งที่มีการเปลี่ยนแปลง (ภายใน 1.5 วินาที)
- มุมบนขวาจะแสดงสถานะ **"บันทึกแล้ว"** เมื่อข้อมูลถูกบันทึกเรียบร้อย
- ท่านไม่จำเป็นต้องกดบันทึกด้วยตนเอง

### 4.8 การดูตัวอย่างแบบฟอร์ม (Preview)

1. คลิกปุ่ม **"ดูตัวอย่าง"** ที่มุมบนขวาของ Form Builder
2. ระบบจะเปิดหน้าตัวอย่างในแท็บใหม่
3. หน้าตัวอย่างจะแสดงแถบสีเหลืองด้านบนพร้อมข้อความ **"ตัวอย่าง (Preview Mode)"**
4. ท่านสามารถตรวจสอบหน้าตาแบบฟอร์มได้ก่อนแชร์ให้ผู้สมัคร

## 5. การแชร์แบบฟอร์ม

เมื่อออกแบบแบบฟอร์มเสร็จแล้ว ท่านสามารถสร้างลิงก์เพื่อส่งให้ผู้สมัครกรอกข้อมูลได้

### ขั้นตอนการสร้างลิงก์แชร์

1. ไปยังหน้า **"แบบฟอร์ม"** (Forms)
2. ค้นหาแบบฟอร์มที่ต้องการแชร์
3. คลิกปุ่ม **"แชร์"** (ไอคอนลูกศร)
4. เลือกระยะเวลาหมดอายุของลิงก์:

| ตัวเลือก | เหมาะสำหรับ |
|---|---|
| **7 วัน** | รับสมัครด่วน หรือเปิดรับระยะสั้น |
| **30 วัน** | การรับสมัครทั่วไป (แนะนำ) |
| **90 วัน** | โครงการระยะกลาง |
| **365 วัน** | การรับสมัครแบบต่อเนื่องตลอดปี |

5. คลิก **"สร้างลิงก์"**
6. คลิก **"คัดลอกลิงก์"** เพื่อคัดลอกไปยัง Clipboard
7. วางลิงก์ใน Email, LINE, หรือช่องทางการสื่อสารที่ต้องการ

> **หมายเหตุ:** ผู้สมัครไม่จำเป็นต้องมีบัญชีในระบบ สามารถเปิดลิงก์และกรอกข้อมูลได้ทันที

> **คำแนะนำ:** ตรวจสอบให้แน่ใจว่าแบบฟอร์มมีสถานะ "เปิดใช้งาน" ก่อนแชร์ลิงก์

## 6. การกรอกแบบฟอร์ม (มุมมองผู้สมัคร)

### สิ่งที่ผู้สมัครต้องทำ

1. เปิดลิงก์ที่ได้รับในเบราว์เซอร์ (รองรับมือถือและคอมพิวเตอร์)
2. กรอกข้อมูลในแต่ละช่อง
3. ช่องที่มีเครื่องหมาย **"*"** จำเป็นต้องกรอก
4. คลิกปุ่ม **"ส่งแบบฟอร์ม"** เมื่อกรอกครบถ้วน
5. ระบบจะแสดงหน้ายืนยันพร้อม animation แจ้งว่าส่งสำเร็จ

### กรณีที่ลิงก์ใช้งานไม่ได้

| ข้อความที่แสดง | สาเหตุ | การแก้ไข |
|---|---|---|
| หน้า 404 (ไม่พบแบบฟอร์ม) | ลิงก์ไม่ถูกต้อง | ตรวจสอบว่าส่งลิงก์ถูกต้องหรือไม่ |
| หน้า 410 (แบบฟอร์มหมดอายุ) | ลิงก์หมดอายุแล้ว | สร้างลิงก์ใหม่และส่งให้ผู้สมัครอีกครั้ง |

## 7. การจัดการใบสมัคร (Responses)

เมนู **"ใบสมัคร"** ในแถบด้านซ้ายจะพาท่านไปยังหน้ารายการใบสมัครทั้งหมด

### 7.1 การกรองและค้นหา

- **ค้นหา:** พิมพ์ชื่อหรืออีเมลในช่องค้นหา
- **กรองตามสถานะ:** New / Reviewing / Approved / Rejected
- **กรองติดดาว:** เพื่อดูเฉพาะใบสมัครที่ทำเครื่องหมายไว้

### 7.2 สถานะใบสมัคร

| สถานะ | ความหมาย |
|---|---|
| **New (ใหม่)** | ใบสมัครที่เพิ่งส่งเข้ามา ยังไม่ได้รับการตรวจสอบ |
| **Reviewing (กำลังพิจารณา)** | อยู่ระหว่างการพิจารณา |
| **Approved (อนุมัติ)** | ผ่านการคัดเลือก |
| **Rejected (ปฏิเสธ)** | ไม่ผ่านการคัดเลือก |

### 7.3 การดูรายละเอียดใบสมัคร (Response Drawer)

1. คลิกที่แถวของใบสมัครที่ต้องการ
2. แผงข้อมูลจะเลื่อนออกมาจากด้านขวา
3. ดูข้อมูลส่วนตัว คำตอบทั้งหมด และไฟล์แนบ

### 7.4 การเปลี่ยนสถานะ

ใน Response Drawer คลิก Dropdown สถานะ → เลือกสถานะใหม่ → บันทึกทันที

### 7.5 การติดดาว

คลิกไอคอนดาว (★) เพื่อทำเครื่องหมายใบสมัครสำคัญ — ดาวสีทอง = ติดดาวแล้ว

### 7.6 การลบใบสมัคร

ใน Response Drawer → คลิก "ลบ" → ยืนยัน 2 ขั้นตอน

> **คำเตือน:** การลบใบสมัครเป็นการถาวร ไม่สามารถกู้คืนได้

## 8. การส่งออกและพิมพ์

### 8.1 ส่งออกข้อมูลเป็น CSV (Excel)

1. ไปยังหน้า **"ใบสมัคร"**
2. ตั้งค่าตัวกรองตามต้องการ
3. คลิกปุ่ม **"ส่งออก CSV"** มุมขวาบน
4. ไฟล์จะถูกดาวน์โหลดอัตโนมัติ

> **หมายเหตุ:** ไฟล์ใช้รูปแบบ UTF-8 with BOM รองรับภาษาไทยใน Microsoft Excel โดยตรง พร้อมการจัดการช่องว่าง (Empty Cells) อย่างถูกต้องเพื่อความสะดวกในการนำข้อมูลไปประมวลผลต่อ

### 8.2 พิมพ์ใบสมัครเป็น PDF

1. เปิด Response Drawer ของใบสมัครที่ต้องการ
2. คลิกปุ่ม **"พิมพ์ / Print"**
3. เบราว์เซอร์จะเปิดหน้าต่างพิมพ์ในรูปแบบ A4
4. เลือก **"Save as PDF"** หรือเลือกเครื่องพิมพ์ → คลิก **"พิมพ์"**

## 9. การออกจากระบบ

1. มองที่แถบเมนูด้านซ้าย (Sidebar)
2. เลื่อนลงมาด้านล่างสุด
3. คลิกปุ่ม **"ออกจากระบบ"**
4. ระบบจะพาท่านกลับไปยังหน้าเข้าสู่ระบบ

> **คำแนะนำ:** กรุณาออกจากระบบทุกครั้งหลังเลิกใช้งาน โดยเฉพาะบนคอมพิวเตอร์ที่มีผู้อื่นเข้าถึงได้

## ภาคผนวก: คำถามที่พบบ่อย (FAQ)

**ถาม: ลืมรหัสผ่าน ต้องทำอย่างไร?**
ตอบ: กรุณาติดต่อผู้ดูแลระบบ IT เพื่อขอรีเซ็ตรหัสผ่าน

**ถาม: สามารถสร้างแบบฟอร์มได้กี่แบบ?**
ตอบ: ไม่มีการจำกัดจำนวน สามารถสร้างได้ตามต้องการ

**ถาม: ผู้สมัครสามารถแก้ไขใบสมัครหลังส่งแล้วได้หรือไม่?**
ตอบ: ไม่ได้ หากต้องการแก้ไข ต้องให้เจ้าหน้าที่ HR ลบใบสมัครเดิมและให้กรอกใหม่

**ถาม: ข้อมูลใบสมัครเก็บไว้นานเท่าไร?**
ตอบ: ข้อมูลจะถูกเก็บไว้จนกว่าท่านจะลบด้วยตนเอง

**ถาม: หากลิงก์แบบฟอร์มหมดอายุ ข้อมูลที่ส่งมาก่อนหน้านี้หายไปด้วยหรือไม่?**
ตอบ: ไม่หาย ใบสมัครเดิมยังคงอยู่ในระบบ เพียงแต่ไม่สามารถส่งใบสมัครใหม่ผ่านลิงก์นั้นได้

**ถาม: สามารถใช้งานบนมือถือได้หรือไม่?**
ตอบ: ได้ แต่สำหรับ Form Builder แนะนำให้ใช้คอมพิวเตอร์เพื่อประสบการณ์ที่ดีที่สุด
